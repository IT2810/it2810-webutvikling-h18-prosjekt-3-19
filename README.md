## Getting Started

### Prerequisites

Expo

```
npm install expo-cli --global
```

### Installing

Installer node pakker

```
npm install
```

Bygg prosjektet med expo

```
expo start
```

Scan QR-koden med mobilen for å åpne applikasjonen.
